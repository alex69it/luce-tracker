# GPS Tracking App

## Overview
This is a GPS tracking app that provides real-time location tracking and displays user locations on a map.

## Features
- Real-time GPS tracking
- User location updates
- Map visualization using Leaflet

## Installation
1. Clone the repository.
2. Install dependencies using `npm install` in both backend and frontend directories.

## Usage
1. Set up the environment variables in `.env`.
2. Start the backend server: `node server.js`
3. Start the frontend: `npm start`

## Dependencies
- Express
- Mongoose
- CORS
- dotenv
